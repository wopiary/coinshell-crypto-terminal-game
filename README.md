**Coin$hell** is a lightweight Python crypto-trading simulator designed specifically to be played within the VSCode terminal. Experience the highs and lows of the market in a retro-style terminal environment.
Disclaimer: This is a simulation game for entertainment purposes only. No real currency is involved, and it does not provide actual financial advice or real-market trading capabilities.


**:: 🛠 Prerequisites ::**
To ensure Coin$hell runs smoothly, you only need the basics:
-Python 3.x: Ensure Python is added to your system PATH during installation.
-VSCode: Recommended for the best terminal display experience.
-Essential Libraries:
  One of the best parts about Coin$hell? No 3rd party libraries are required. The game runs entirely on Python’s standard built-in tools, so you don't have to worry about pip install errors.


**:: 🚀 Getting Started ::**
Follow these steps to get Coin$hell up and running:
1.) Download the Project: Download the repository as a ZIP file and extract it to your preferred location.
2.) Open in Editor: Launch the extracted folder in VSCode.
3.) Open Terminal: * In VSCode, go to Terminal > New Terminal.
    Alternatively, use your operating system's native Terminal and navigate to the folder.
4.) Run the Game: Execute the script using:
     Bash
      **python crypto_trading_terminal_game.py**  
      
Enjoy the ride!
